PORTFOLIO

virtual phone chatting

This web-based AI chat system aims to provide emotional relief for those experiencing loneliness, stress, or anxiety due to long-term solitary living. By simulating the existing social chat app WeChat as closely as possible to create a real-life experience, the AI simulates real-life chat and interaction. This allows users to believe they are speaking with a real person, providing them with the opportunity to confide in others, a feeling they lack in the real world, and thus alleviating stress. Within our virtual phone, users can engage in individual one-on-one and group chats, browse forums, and interact with messages within them. During chats, they can share music or transfer money with the AI, which will respond and reply accordingly. Voice and video chats have also been added, simulating text. The API used is the LLM API, maximizing the fidelity of real-life chats. For privacy reasons, we've decided to allow users to connect to the system using our own API to ensure privacy. We'll provide a dedicated API and pre-created AI characters for a preliminary experience. Due to the complexity of the system, ETHAN is responsible for building the HTML and CSS, while Jessica is responsible for debugging the JS and API.

emotional visualization

This interactive kaleidoscope visualization creates an immersive emotional experience through speech recognition and dynamic visual response. The project detects five key emotions - happy, sad, angry, peace, and boring - through voice input, transforming each emotion into a unique visual representation using color palettes and particle effects. When users speak these emotion keywords, the system responds with color transitions that reflect emotional states: warm yellows and oranges for happiness, cool blues for sadness, vibrant reds for anger, calming greens for peace, and muted grays for boredom. The design features a 3x3 grid of kaleidoscope patterns that rotate and transform, complemented by particle effects that burst across the screen during emotional transitions. The experience begins with a simple button prompt, inviting users to interact through their voice while creating a hypnotic visual landscape that evolves with their emotional expression. This project demonstrates how sound input can create meaningful visual feedback, bridging the gap between verbal expression and aesthetic representation.
